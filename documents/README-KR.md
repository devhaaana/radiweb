## 경고

**이 프로젝트를 상업적인 용도로 사용하지 마십시오. 개인적, 비상업적인 용도로만 사용해 주세요.**

## 사용 기술

- `HTML`
- `CSS`
- `JS`

## 기술적 세부사항

PC(HTML5) 버전의 Radiko는 사용자의 위치를 IP 주소를 통해 인증합니다. 그러나 Android 버전의 Radiko는 IP 주소가 아닌 GPS 정보를 기반으로 사용자를 인증합니다.
